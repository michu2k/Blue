// How to install ?

	1. Move catalog with theme to the `/themes` 
	2. Active theme in administration panel 

// Homepage

	1. Select Pages -> Add new
	2. Insert title and content
	3. Select home.html template
	4. Change homepage in settings

// 404

	1. Select Pages -> Manage
	2. Select 404 page
	3. Change template to 404.html

// Sidebar

	Sidebar content uses: {$snippet.widget-content}. 
	You need to create snippet if you want to change your content.  

// Social icons

	You can modify your social icons in the sidebar.

		<a href="#">
			<i class="fa fa-facebook" aria-hidden="true"></i>
		</a>

		# - Link
		fa-facebook - icon name

	Icons: fontawesome.io/cheatsheet/
